#Frontend — Sadqa Karu

This folder contains all frontend (UI) code for the Sadqa Karu donation platform.  
The frontend is built using React.js / Next.js with Tailwind CSS for styling and Axios for API communication.


#Tech Stack
- React.js / Next.js  
- Tailwind CSS  
- Axios  
- React Router / Next Router  



# Goals
- Develop all static UI screens based on Figma designs  
- Make pages responsive (mobile, tablet, desktop)  
- Use mock JSON data until backend APIs are ready
